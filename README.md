Simple parallax mouse move library w/orientation movement.

#### Install:

`npm i cidi-mouse-move` or `yarn add cidi-mouse-move`

#### Import:

`import { CidiMouseMove } from 'cidi-mouse-move';`

#### Use:
```
<CidiMouseMove distX='0.02' distY ='0.02' oMult='3' time='0.5'>
  <img style={{
    width: '300px',
    height: '300px'
  }} src={img} alt='test' />
</CidiMouseMove>
```
